### Dependencies

- ``python``
  - https://www.python.org/
- ``moviepy``
  - https://zulko.github.io/moviepy/
- ``ffmpeg``
  - https://zulko.github.io/moviepy/
- ``scoop``
  - https://scoop.sh/

currently tested on ``windows``

### Usage

```
python mp4tom4a.py input_file.mp4 output_file.m4a
```

### Quickstart

On Windows:

```
pip install moviepy
```
```
scoop install ffmpeg
```

On Linux:

- Install the deps with your distro's package manager. :)
